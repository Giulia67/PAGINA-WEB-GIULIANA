Se vio qué hace el CSS?

Permite crear reglas que especifican la apariencia del contenido de un elemento en HTML. Se puede modificar la funte, color de texto, tamaño de texto, posición.

Que mira CSS en HTML? Todo esta adentro de una caja. Cada caja engloba un elemento o agrupación de elementos.

Uno de ellos es colores , la propiedad de color permite especificar el color de texto en 3 formatos:

-Valores RGB

-Códigos HEX

-Nombres de Color

Background-color: esta declaración rellena la caja dende está contenido el elemento o elementos de HTML. Se debe considerar los formatos mostrados en el slide de color.

Opacidad : permite de manera sencilla agregar opacidad a un elemento así sobreponemos colores si fuera necesario para nuestra aplicación.

Colores HSL (Hue,Saturation,Lightness)

HUE: configura el tono ddel color desde los 0 a 360 grados (default:0) Saturation: configura la cantidad de saturación de grises en un color. Se representa de 0 a 100% Lightness: configura la cantidad de luminosidad en el color. Donde 0% es oscuro, 100% es iluminado y 50% es normal.
